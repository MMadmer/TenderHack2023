Рисунок 254 – Действия Поставщика с Заявкой по потребности – поиск и ознакомление
1. Перейти на страницу закупки одним из способов:
из раздела «Уведомления», для этого рекомендуется настроить фильтр получения
уведомлений;
в  едином  реестре  закупок  (раздел  7)  перейти  в  подраздел  «Закупки  по
потребностям»,  настроить  фильтр  и  перейти  на  страницу  закупки,  нажав  на
наименование закупки по потребностям. 
Закупка должна быть в статусе «Прием предложений».

Общая информация (Рисунок 255 (1)):
