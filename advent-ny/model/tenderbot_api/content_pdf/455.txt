Инструкция по работе с сервисом Логистика

Приложение 1.  Ввод адреса с использованием 
справочника ФИАС  
Для ввода адреса с помощью справочника ФИАС необходимо выполнить следующие 
шаги:  
1. Для перехода в режим редактирование необходимо нажать на кнопку «Изменить» 
(Рисунок 33) 
 
Рисунок 33 – Кнопка «Изменить»  на форме ввода адрес с использованием 
ФИАС  
2. В появившейся строке начать вводить название региона или воспользоваться 
поиском по справочнику нажав на значок лупы (Рисунок 34)
