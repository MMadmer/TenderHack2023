Инструкция по работе с сервисом Логистика

 
6 Реестр «Мои заказы по доставке»  
Созданные пользователем з аказы по доставке можно посмотреть в реестре «Мои 
заказы по доставке» . Для этого необходимо перейти в личный кабинет, выбрать раздел 
меню «Сервисы», подраздел « Логистика » и перейти на  вкладку «Мои заказы по доставке» 
(Рисунок 31).  
Реестр состоит из реестра с карточками заказов и блока с фильтрами.  
Для перехода на форму просмотра заявки на доставку необходимо нажать на 
наименование заявки ( Рисунок 31 (1)). 
Для отображения блока с фильтрами необходимо нажать на кнопку «Показать 
дополнительные фильтры»  (Рисунок 31 (2)).  
 
Рисунок 31 – Реестр «Мои заказы по доставке»  
В реестре возможен поиск по следующим критериям  (Рисунок 32): 
⎯ Наименование заказа  (поиск по наименованию) ; 
⎯ Сумма заказа от…до  (поиск по сумме заказа ); 
⎯ Заказа с …по (поиск по дате  создания  заказа ); 
⎯ Номер заказа ; 
⎯ Транспортная компания (выбор из выпадающего списка) .
