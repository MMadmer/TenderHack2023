В полях «Контактный телефон отправителя» и «Контактный телефон получателя» 
важно указать актуальный номер человека, который будет вза имодействовать с 
логистическим оператором. Менеджер логистического оператора будет связоваться по 
указанным номерам для подтверждения заказа.  При отсутствии ответа по указанным 
контактным номер перевозка может не состояться.
