1.4.3 Элемент  curren  cies 
Элемент  currencies  задает список курсов валют магазина. Каждая из валют 
описывается отдельным элементом  currency.  
Данные о ценах принимаются в рублях (RUR, RUB ).  
Пример : 
<currencies > 
  <currency  id="RUR" rate="1"/> 
</currencies > 
Атрибут  id элемента  currency  указывает код одной или нескольких валют, которые 
могут быть использованы в YML -файле.  
Атрибут  rate указывает курс валюты к курсу основной валюты, взятой за единицу 
(валюта, для которой  rate="1"). Параметр  rate может иметь следующие значения:  
• Постоянное число  — внутренний курс, который использует поставщик ; 
• CBRF  — курс по Центральному банку РФ ; 
• СВ — курс по банку той страны, к которой относится магазин по своему 
региону, указанном у в личном кабинете.  
В качестве основной валюты (для которой установлено  rate="1") могут быть 
использованы только рубль (RUR, RUB) .
