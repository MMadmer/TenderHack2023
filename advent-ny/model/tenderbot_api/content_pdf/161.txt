Инструкция по работе с Порталом для поставщика

248
Блок  «Достижения».  Отображаются  все  достижения  заказчика.  Для  просмотра
полного списка достижений, которые еще не получены заказчиком необходимо
нажать на «Подробнее»;
Блок «Контакты». В блоке отображается контактная информация компании;
Блок «Связаться с нами». Блок доступен только авторизованному пользователю. В
блоке отображается имя и Email авторизованного пользователя и поле сообщение -
для отправки вопросов заказчику.
