15Реестры поставщиков и заказчиков
На Портале поставщиков существует возможность просмотра информации о заказчиках
и поставщиках.
15.1Реестр поставщиков
Для  просмотра  информации  о  поставщиках  пользователю  необходимо  перейти  в
реестр  поставщиков.  Для  поиска  поставщиков  необходимо  в  Главном меню  Портала
поставщиков (Рисунок 21) нажать на «Организации» → «Поставщики».

15.1.1Критерии поиска поставщиков
Поиск  поставщиков  осуществляется  по  одному  (или  нескольким)  из  следующих
критериев (фильтров), указать которые можно в правой части раздела по кнопке «Показать
фильтры» (Рисунок 339) (затем можно нажать кнопку «Показать расширенные фильтры») :
