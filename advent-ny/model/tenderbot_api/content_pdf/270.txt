2.3.Блок «Дополнительные сведения о компании»
В блоке «Дополнительные сведения о компании» пользователю необходимо заполнить
обязательные поля указав дату присвоения ОГРН и выбрать тип системы налогообложения
(Рисунок 151).
Рисунок 151 – Блок «Дополнительные сведения о компании»
2.4.Блок «Участники принципала»
В блоке «Участники принципала» пользователь может по запросу банка добавить в
заявку информацию о сотрудниках ( Рисунок 152) или организациях учредителях ( Рисунок
153), для этого необходимо нажать на кнопку «Добавить».
 В  заявках  некоторых  банков  требуется  добавить  информацию  о  сотрудниках  или
организациях учредителях.
