2.5.9Встроенный календарь
Встроенный календарь  используется при заполнении значений реквизитов, имеющих
строго определенный формат даты. Для того, чтобы выбрать значение с помощью календаря,
необходимо нажать на кнопку  
 . По умолчанию встроенный календарь открывается на
текущей дате (Рисунок 15).
