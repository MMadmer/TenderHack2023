Инструкция по работе с машиночитаемыми доверенностями

 
Рисунок 2 – Добавление МЧД  
 
 
На открывшейся странице пользователю необходимо загрузить доверенность, 
добавив xml-файл в поле «Загрузить МЧД из файла» ( 2 (1)) и нажать на кнопку «Добавить» 
(Рисунок 2(2)). 
Добавленная доверенность отобразится в таблице, и пользователю станут доступны 
следующие действия:  
 кнопка «Удалить» ( Рисунок  (1)) - при нажатии удаляет выбранную 
доверенность;  
 кнопка «Скачать xml МЧД» ( Рисунок  (2)) - при нажатии скачивает файл 
доверенности в формате *.  xml; 
 кнопка «Обновить» ( Рисунок  (3)) - при нажатии обновляет статус выбранной 
доверенности, если доверенность была отозвана, то статус поменяется;  
 номер доверенности ( Рисунок 3(4)) – при нажатии открывает модальное окно 
«Полномочия» ( Рисунок ) со списком полномочий выбранной доверенности;  
 Удалить можно только ту доверенность, которая ни разу не использовалась.
