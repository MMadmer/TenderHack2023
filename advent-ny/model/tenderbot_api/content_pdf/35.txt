Рисунок 22 – Отображение всплывающего меню
В левой части всплывающего меню  содержатся разделы, в правой части - подразделы:
Раздел  «Каталог» предназначен  для  перехода  на  страницу  выбора  категорий
товаров,  работ  или  услуг  при  нажатии  на  конкретный  подраздел  откроется
страница поиска наименований в соответствующем подразделе каталога:
Подраздел «Товары»  предназначен для перехода на страницу поиска товаров;
Подраздел «Работы»  предназначен для перехода на страницу поиска работ;  
Подраздел «Услуги» предназначен для перехода на страницу поиска услуг.  
Раздел «Планы и планы-графики» предназначен для перехода на страницу поиска
планов:
Подраздел  «Все  планы»  предназначен  для  перехода  на  страницу  поиска
планов;
Подраздел «Планы-графики»  предназначен для перехода на страницу поиска
планов с фильтром по план-графикам;
Подраздел «Планы закупок»  предназначен для перехода на страницу поиска
планов с фильтром по планам закупок;
Раздел «Единый реестр закупок» (см. раздел 5):
Подраздел  «Все  закупки»  предназначен  для  перехода  на  страницу  поиска
закупок на вкладку всех закупок;  
Подраздел «Котировочные сессии»  предназначен для перехода на страницу
поиска на вкладку котировочных сессий;  
Подраздел  «Закупки  по  потребностям»  предназначен  для  перехода  на
страницу поиска на вкладку закупок по потребностям.  
Подраздел  «Конкурентные  процедуры»  предназначен  для  перехода  на
страницу поиска на вкладку закупок по конкурентным процедурам;  
Подраздел «B2B-закупки»  предназначен для перехода на страницу поиска на
вкладку закупок B2B;
