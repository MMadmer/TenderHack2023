Рисунок 145 – Заявка на регистрацию банковской гарантии
 В  форме  «Заявка  на  регистрацию  банковской  гарантии»  часть  полей  предзаполнены
информацией из профиля пользователя.
Для перехода к редактированию данных заявки необходимо нажать кнопку «Внести
данные» (Рисунок 146).
Рисунок 146 – Переход к редактированию заявки
 Перед отправкой заявки необходимо заполнить все обязательные поля.
В зависимости о выбранного банка набор обязательных полей может отличаться.
2.1.Блок «Данные по заявке»
В блоке «Данные по заявке» отображаются наименование банка и предзаполненные
данные пользователя ( Рисунок 147). Пользователь может отредактировать и внести данные в
обязательные поля блока.
