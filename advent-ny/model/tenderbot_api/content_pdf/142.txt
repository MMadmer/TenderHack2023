Инструкция по работе с Порталом для поставщика

Рисунок 319 – Блок «Сведения о поставке»
При нажатии на кнопку «Добавить исполнение» отобразится окно с выбором этапа
контракта и типа документа ( Рисунок 320). Доступно к созданию три типа документа: счет-
фактура, счет-фактура и документ о приёмке (УПД статус 1), документ о приёмке (УПД
статус 2).
Рисунок 320 – Выбор этапа и типа документа исполнения
Для  сохранения  выбранного  типа  исполнения  в  качестве  значения  по  умолчанию
нажмите «Запомнить выбор типа документа» в правом нижнем углу модального окна.
Изменение  типа  исполнения  в  созданном  исполнении  невозможно.  В  случае
ошибочного выбора типа создайте новое исполнение.
В  результате  добавления  сведений  об  исполнении  в  контракте  появится  блок  с
наименованием «Исполнение» ( Рисунок 321).
